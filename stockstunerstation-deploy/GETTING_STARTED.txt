STOCKSTUNERSTATION - START HERE (beginner guide)
===============================================

ON YOUR PC (easiest - do this first)
------------------------------------

1. Double-click:  START.bat
   (First time it installs everything automatically.)

2. Wait for the black window to say "Running on http://127.0.0.1:5050"
   Your browser should open by itself.

3. Use the dashboard:
   - Left menu: Unusual activity, Gainers, News wire, etc.
   - Click "Refresh scan" after 1-2 minutes if the table is empty.

4. To stop: close the black command window, or run stop_dashboard.bat

5. To start again later: double-click START.bat


IF SOMETHING GOES WRONG
-----------------------

- Blank page or "URL not found":
    Run stop_dashboard.bat, then START.bat again.
    Use the URL printed in the black window (not an old bookmark).

- "Python is not installed":
    Install from https://www.python.org/downloads/
    Check "Add python.exe to PATH" during install.
    Then double-click install.bat, then START.bat
